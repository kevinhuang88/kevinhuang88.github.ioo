# 前言
此文之思由来已久。其所载之事非奇，然所寄之情则深。昌黎公尝语，文宜为时而著，其时乃余之将弱冠。弱冠者，将以冠决天下，然势弱，无提携则不可也。此文所记乃提携之事。有子问于鄙人：汝何以知汝之未来？余大骇，不知何以应，乃复之：余不曾知也，然定不止于仰天大笑出门去。何老师所言亦如是。谨布此文于此时，以录余志。龙溪黄琮熹，美国东部时间乙亥七月丙子日，子时三刻。  

# 一
夜晚，房间里的空调机嗡嗡响。小黄躺在床上，望着天花板上的灯。这时，他的手机响了，是一条新闻推送。“本市地铁二号线已经开始空载试运行，预计今年年底将投入使用……”他看着这条消息，嘴角露出微笑。离他家几步之遥的地方，就是地铁站；地铁的另一端，就是他记挂的地方。小黄搬到这个地方已然两年，两年的分别对他而言不可谓不长。不过，即使小黄不搬家，这份分别的忧伤也是不可避免的——  

他已经不在这座城市继续学习了。  

窗外，华灯璀璨，高架、隧道内车流络绎不绝。马路的另一侧则是另外一个世界：精致的小房子整齐地排列于岸边，海湾内部没有一丝波澜。再往里走，则是另一处有些杂乱的聚落，几盏LED灯默默地挂在墙外。十年前，这里还全都是塍、厝和庙，只有一条水泥路通向遥远的天边。十年前，小黄住在另一片水域的边沿；他想，十年前何老师也像他现在一样，远离家乡，在两千公里外的另一片水域边上学习——当然，这片水五十年前就不复存在了。天气晴朗的时候，小黄乐意在这段路上散步。散步的时候，他总是感叹，白色的灯光不如路灯的黄色灯光来得宁静。  

# 二
在一个有些闷热的晚上，太阳尚未完全落山。小黄和小海刚刚吃饱，他们并排走在山腰的小路上。小路随山势蜿蜒，向前方深入。小黄和小海在路上走着，有一搭无一搭地聊着学业的话题。他们走到了一个铁门，道路在这里分岔。以往，他们走到这里都会原路返回，赶紧走回家去写作业。但是，今天，小黄对小海说，你有空吗，不然我们往上走吧。小孩说，好呀。于是，他们的身影便消失在铁门后，消失在雨后的浓雾中。  

这是三年前的事了，不过现在小黄也仍然经常想起它。小海现在也在上大学，和小黄离得不远。每当小海和小黄一起散步的时候，小黄都会说，你记得我们爬山的那一天吧。此时，小海总是会用手抽打小黄的背部，然后娇声嗔道，讨厌。小黄笑了，说，你好像何老师呀。小海说，哪里，你说反了。  

小黄在路上散步的时候，喜欢放飞自己的想法。何老师是他心里最经常想的人之一。他想，只要走过那片水，再穿过那个村，就可以看到何老师了。自从三年以前何老师结婚以后，他就一直住在那里。但小黄从来没有去找过何老师。每次小黄都是走到村子内的一个四岔路口，然后原路返回。我想，小黄之所以没有去找何老师，可能是因为没有小海陪伴吧。或者也可能是小黄从内心里对何老师就有天然的抗拒——这个说法比较奇特，但是根据心理学的原理，事实可以如此。  

十年前，何老师在他的宿舍楼里，望着斑驳的墙面和铁床的栏杆，也在想着他什么时候可以找到对象。何老师认为，爱情是求之不得的，“缘分不停留，像春风来又走，”什么时候可以找到自己的另一半，全靠命。何老师当然没有预测未来的能力，他也不知道他当了老师以后，他的情感生活会是如何翻天覆地。  

# 三
“找我做什么？”昌毅从椅子上缓缓站起来，对我望着。“老师，”我缓缓道，“我每天都在做着一样的事情，永远都在为几张纸而奋斗。从前我可以将这几张纸同远大的前程联系起来，但现在，这种微弱的联系已不足以支撑我的奋斗历程了。”“写了就是了，”昌毅说，“不要想太多。”“写不下去了。不知道为什么写它，怎还能写得下去呢？”  

昌毅起身，向房外走去。我跟着他出门。屋外，月色微醺，走廊灯火明亮。  

“我们来谈实际情况。即使你不想写作业，不想学习，但是你总得接受检验，总得参加高考。你既然选择了在这学校里学习，这条路就是你唯一的出路。我们不说别的，到时候你一旦考砸了，怎么办呢？目标驱动当然很好，但是没有目标，我们是不是应该从结果考虑一下呢？”  

我对着昌毅出神地看着，好像小黄第一次出神地看何老师一样。那是在之前的一个夏天，一间比较大的教室——何老师穿着一袭和他黑色的皮肤相映衬的黑色衣服，在讲台上讲课。小黄说，何老师的眼神给人一种凶神恶煞的感觉，但这无法冲淡他对他的喜爱。这种爱是发自内心的，由心间的联系而引起，不会轻易结束的。现在，我已经无法回忆我对昌毅的感觉；但这不重要。即使有了这种感觉，谁也不会知道三年、五年以后这份情感是否仍会持续。  

# 四
人的记忆是会产生变化的。其实，那天昌毅可能没有跟我说这些话，也可能跟我说了这些话；又或者，根本没有昌毅这个人，这些都是我在梦里自作多情搞出来的。小海告诉我，小黄就经常在梦里虚构一些人物。比如，在那个夏天的夜晚，他闭眼睡觉以后，就梦见一个人从地铁轨道的另一头走来。他和何老师一样高，比较壮实，脸略圆，五官和何老师也很像。他走到了小黄的跟前，轻轻地抚摸他的头。“你是谁呀？”小黄问道。“我是你常常想的人。”“何老师！”“不，我不是何老师。我代替何老师来看你。”“可你讲话的声音和何老师一样啊。”“那是当然。我就是你心里何老师的模样。”小黄的心头顿时涌上一股酥酥的暖流。“何老师，我想问你一个问题，一个驻留我内心已久的问题。”  

“你爱我吗？”  

“哈哈哈！”小海每次讲到这个地方的时候，都会不自觉地发笑，“真可怕！何老师早就结婚了，而小黄也知道。但他就问这些愚蠢的问题！”“万一小黄想说的是，他喜欢你呢？”“怎么可能！”小海捧腹，“他还喜欢我？要是他喜欢我，他早就告诉我了！”  

小海跟我讲这些话的时候，是一年前，在我房间的窗边。那一天，天空的云出奇的少，正如我和昌毅在操场散步的时候。我和昌毅散步的每个晚上，天上的云都很少。我们绕着四百米跑道一圈又一圈地走着，我讲三句，他讲一句。我说家国大事，他说学校杂事。那时我常常希望我们就这样一直散步下去；但是，晚自习的铃声总催着我回到教室。  

然而晚自习的铃声无法阻止小海从办公室里出来。  

# 五
十年前，何老师在从前的北太平湖旁边学校宿舍里住着的时候，他不会想到，几年后有一个叫小黄的他的学生对他坠入爱河。当他在冬日吹着西伯利亚刮来的狂风时，他想的是，什么时候我才能回到温暖的南方。几年以后，当他躺在舒服的弹簧床上，睡在他夫人边上的时候，他却在想，吹一点冷风也是不错的，不要被一时的爱情冲昏了头脑。早上起来，何老师又坐上地铁，朝学校进发。作为一名班主任，职责规定了他必须在学生到教室前就到达班级。  

何老师到班级，看到小黄的座位总是空的。小黄从来没有在上课铃响后十分钟内进过教室的门。应该好好管教一下他了，何老师想，这种做法是违反校规校纪的。但是，何老师却觉得，他搞不清小黄的心理。小黄经常来找他聊天，说家国大事，也说一些可笑的他的内心波澜。班主任和学生谈话总是天经地义，但何老师发现，小黄看他的眼神总有一些特别之处：这种眼神使何老师的心中不由得升起一丝爱怜。  

在另一个夏天里，小黄站在窗前，望着太阳和月亮。他仿佛已经在这里站了好几年，从何老师来，到何老师离开，再到他离开这座城市。窗外，星辰渐稀，远处聚落和海湾的灯若隐若现。小黄想，那一盏盏路灯，是不是何老师在他心中的投射呢？小黄想起了他的梦，想到了梦里的那个人。他把那个人命名为小史。史者，其音似思，小黄欲以此寄情。  

# 六
我的这个故事还有另一个版本。在这个版本中，小黄没有和何老师发生任何关系。相反地，小黄和小史成为了莫逆之交。小史有着和小黄一样的特点和爱好，就像我和昌毅一样,可以聊任何事情。经常可以看到小黄和小史傍晚时分在操场上散步，他们沐浴在夕阳的余晖里，背影甚是好看。  

在这个故事中，小黄仍然经常站在家里的窗边，望着海湾和聚落，还有更加遥远的地方。我们不知道他在想些什么。或许是小史？但小史的家远在千里之外，他根本无法看到。又或许是小黄和十年前的何老师一样，想着他未来的爱情和家人？我无从得知，因为小黄、小海、小史都不能告诉我答案。  

不过，小黄可以告诉我三年前的一段故事。他说，他和小海朝山上走去的时候，路灯的光渐渐被散射了，道路上的落叶也变得多了起来。鸟和蝉鸣的音量也不断升高，不过并没有什么奇怪的人或物体。随着他们向上走去，两人的话也变少了，他们开始顾着走路，任凭寂静笼罩这个世界。正当他们走到一个三岔路口的时候，突然，路灯灭了，世界陷入黑暗。  

小黄把故事讲到这里就停止了。我曾经试图把这个故事代入其它小说家的思路，但发现遇到了大麻烦：按照他们的思路，小黄要么不能活着回来，要么就是神仙，要么会留下一些后遗症——而这些都是不存在的。这只能证明：要么这个小黄和小海的故事是编的，要么小黄是外星人。  

# 七
随着我和昌毅交情的日益深入，我们所讨论的话题也日益丰富了。我还记得，有一个夏天，正是校学生会竞选的前夕，我和昌毅讨论各个候选人的竞选纲领。
“我感觉，这些竞选纲领的原则性太强，实际措施几乎没有。而且，这些候选人似乎营营于介绍他们自己的气质，而对它们的领导能力没有任何提及。”  

“这是正常的。你想想，在一个学校内，学生会能办的事寥寥无几，学生会主席的权限也是受限缩的。你只需要挑一个你最满意的人当主席就可以了。”  

“我们学校招生的时候还大言不惭学生享有充分的自由，看来此言甚假。”  

“没有绝对的自由，”昌毅望着我的眼睛，“你走向社会就知道了。”  

那个晚上，恰逢望日，月光如水。在本故事中涉及的其它几个晚上，都没有这样好的月光；而在那时的我看来，月光是冷的，是一种清冷。那时，我每天一旦有时间，就会走进昌毅的办公室，理由是：教室没有空调，太热了。小海说，当时有同学议论我，说我怎么每天都去找昌毅吹空调，去哪里吹不行？我说，那是因为昌毅的办公室就在我们班隔壁。如果你一定要问我背后的真实原因，我只能回答：我也不知道。  

就像小黄说不清他和小史、何老师之间的感情一样。  

# 八
三年前，小黄和小海在一条山路上散步。他们走到一个铁门处，然后又向上走。三年后，小黄和小史在操场上散步。他们一圈又一圈地走着，好像没有一个终点。  

# 九
小黄走在通往海湾的路上。他一边走，一边想，何老师到底在何方。他的心中浮现过何老师与他在一起时候的一幅幅画面。何老师好像站在他的身旁，轻声细语地问他心中的想法，他温柔的话迫使小黄掏心置肺；何老师又好像站在他的前方，他恨不得直冲上去，就像他看到小史的时候一样。小黄又想起了何老师在北太平湖边上的时光。他非常想知道何老师当时心中在想些什么，但奈何他根本没有途径知道——何老师从不会告诉他这些。  

小黄想对何老师说：我爱你。  

但在这句话说出口之前，他就永远失去了价值——因为何老师结婚了。他不愿横刀夺爱。  

小黄也想对小史，对小海说：我爱你。但它们都不如对何老师的来得真诚。爱的人可以有千千万万，但真心相邀的人只有一个。  

# 十
我还记得毕业前我和昌毅在一起的最后一天。我们静悄悄地走在回家的大坡上。我很快就要搬离这个地方，昌毅也很快要乔迁新居。在我们要分开的路口处，我说：“今天的月亮很圆。”他说：“树叶的影子很美。”  

不知道小黄在从楼上向下望的时候，有没有类似的体会。
