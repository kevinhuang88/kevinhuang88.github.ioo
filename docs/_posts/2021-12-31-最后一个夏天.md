# 一
婆娑的树影掩映着水泥地面，与隔壁大楼伸出的雨棚的阴影相连接。树影之外则是阳光的舞台，它直扑大楼的白色外墙，给空气更增添一份热。在这样一个普通的午后，我带着帽子和墨镜，在路上行走。楼房、树木和雨棚的阴影之间有时会有间断，我不得不灵活地移动我的身体，以使我尽量少受到太阳的照射。这是近一段时间以来少见的一个晴天。若是下雨，仅凭这一点点雨棚则不能避免我被淋成落汤鸡了。

远处，一栋栋被刷成红、橙、蓝等各色的大楼在远处矗立，与眼前的绿色草坪一起，构成一幅和谐的图画。路上不时有一两辆汽车驶过，带来一阵声音，然后整条路又陷入沉寂。这一场景使我回想起我小时候在小区公园草坪上玩耍的场景。每到像今天一样炎热的天气里，妈妈每次都会在我的背后塞上一块纱布，以使我的汗不至于浸湿衣服。

今天，看起来就像小时候普通的一天；可我脚下踏着的，已不是家乡的土地。这里是星岛，一个永远被困在夏日的地方。

# 二
我是从萧瑟砭骨的大漠边缘，一蹴而至大陆极南的海角的。换句话说，我一步就从冬天进入了夏天。当然，这句话说得也不完全正确：在我动身的时候，太阳也正把婆娑的树影和发热的水泥地带给朔方大漠。在这样的天气里，我穿着两件厚重的长袖衣服，挣扎在烈日之下，移动在学校内不同的景物之间。我试图以少喝水延缓衣服变湿的速度，但根本无济于事——幸好我最终没有中暑。这主要归功于北方夏日的干燥。如果我在南方，很有可能就躺在医院的床上了。

北方的冬天，既有随风满地石乱走，也有千树万树梨花开。这一年的冬天，前半段时间是被罕见的寒冷笼罩的。就在那个最冷的晚上，我坐在电脑前，参加了一场为时半小时的面试。孱弱的暖气根本不能敌从缝隙灌入的狂风，我因紧张而发热的身体也远不能温暖我的手脚。半小时的面试过后，我仍然在两件毛衣和保暖内衣的护卫下打冷颤。两周以后，当我回到家里而天气不那么冷的时候，收到拒信一封——马上又把我的心情打入冷宫。当时的我不会想到，这只是被不断拒绝的开始。

在整个冬天里，我都在被面试、被拒绝的循环中度过。虽然原本不爱发朋友圈，但在这段时间，我一共发了三十条。为了不让我的朋友认为我疯了，我还为每一条朋友圈精心设置权限，保证他们不会高频率看到。在这个冬天后半段高于均值的温暖中，我却不断在想如果我延期毕业一年，生活会有什么不一样；或者在思索，我是否将永远被困在在这个冬天里呢？

# 三
今年，那个小区因危房改造，被整体拆迁。随着挖掘机铲子重重地落下，旧楼房化作一堆瓦砾和尘土；我从小到大在这里生活的场景，便就此封存在我的回忆里和照片中。不知道我完成学业的时候，新的小区能否完成建设呢？规划里说新的小区会保留一些旧小区的痕迹，不知道是真的还是安慰剂呢？

在有关生活的回忆里，我印象最深的就是小区的公园。小时候，我既怕热，也怕冷。南方的冬天没有暖气，十分冻人。每到冬天有太阳的时候，我都会和家里人一起去花园里散步。柔和照射到厚棉袄上的太阳缓缓将热量带进体内深处，让人感觉非常舒服。我当时不会想到多年以后的冬天，在冰河的岸边散步的我，会讨厌照在身上使全身发热而不得不让人取下羽绒服的艳阳；而当我在冰河的岸边散步的时候，也不会想到仅仅一个月以后，我会坐在南方的家里，懊悔我这一愚蠢的想法。

我散步的习惯是从小就养成的。但大漠边缘的狂风扼杀了我的这一习惯；只有在晴天的中午或者冷空气间歇而且不起雾的晚上，我才能在饭后漫步河边。河边的路是两年前重新铺过的，平时走的人也很少，适于我慢慢徘徊。在那个冬天，我走在河边的时候，脑海里想的一般是：最近什么人又要给我面试？我向什么人请求面试了？还要给谁请求？这些面试、材料和风雪一起，绘制了这个冬天的灰白底色。

# 四
夕阳薄暮，天色渐晚。微风拂过，我走在从实验室回家的路上。我的实验室不在系馆。有空的时候，我都会从实验室散步回家——这个时候，恰是黄昏，天气不热，正适合步行。在我的左手边，一栋教学楼矗立在小山坡上；右手边则是一片植满草的斜坡，斜坡的下面是几栋矮房子。和路上的房子不同，这几栋楼房的外墙以白色为主，以红瓦作屋顶，有很浓厚的传统建筑风格。虽然这里没有河，但散步是不拘泥于场地的。

星岛的气候，完全没有任何时间方面的参考价值；即使是每年信风带来的晴雨变化，也只是在一个很高的降水量基础上进行小幅增减，在我看来就只是多雨而已。比如，就在我写作这句话的时候，窗外的天便是灰白色的。或许不过多久，它就会转暗，然后就会有一阵雨倾泻下来，把正在路上走的人浇个透心凉。当然，穿得少一些可以很好地避免雨把袖子和裤管打湿。

穿少一些的理论也适用于宿舍里。当我感觉宿舍里很热而又不想开空调（为了省钱）的时候，就会少穿甚至不穿衣服，像我在家里一样。前几天，我在宿舍褪腹裼的时候，就恰好收到一个同学的短信，说有一个女朋友要介绍给我认识。——于是我吓得立刻把衣服给穿了起来。我竟然已经二十二岁了，已经到了谈婚论嫁的年龄。不管怎么说，今年的冬天和夏天，已经不可能再回去了。很久以后，星岛永远的夏天，也会像楼房的残垣断壁一样，就此封存在我的回忆里和照片中。

# 五
北方的春天是短暂的。对我而言，每到惊蛰或春分的时节，停止供应的暖气便是春天即将到来的信号；之后，树木逐渐抽丝吐绿，天气也变得不那么冷，一直到四月（农历），便可以全天着夏装了——对我而言，也就是夏天到了。春天，也意味着我散步的时间可以变得更长，不会因为随意的狂风或冷雨雪而被困在室内。而今年的春天，我的心境却仍然不太平。

面试、面试、还是面试。面试从冬天贯穿到夏天。每一个面试前，我都读坐屏幕前，把有关内容看一遍又一遍，然后在下一次面试前忘掉，然后再看一遍又一遍；每一个面试后，我都汗流浃背，不得不下楼河边走几步。随着天气变热，我每一次面试以后，流的汗就多一点。

一直到树叶完全长出，夏天到来，我才最终知道，接下来的数年我将会在星岛度过。接到这个消息的时候，想到我马上就要和夏天缠绵很多年，我不禁打了一个寒战：小时候，我每到夏天都会容易地流一身汗。比如在洗澡以后，五分钟之内我就会汗流浃背，令刚刚洗好的澡几乎失去其作用。但后来我想到我在北方的冬天里皮肤皲裂，每天早晨足足花十分钟抹护肤品的经历，我便略有了一丝安慰。反正我又不爱出门逛街，我就不信在实验室还能搞出一身汗。所以，即使我在离开北地的时候差点中暑，我也不对我的想法有任何疑问。

当我最后踏上前往星岛的航班的时候，我也就彻底地与冬天告别了。

# 六
小时候，我住的房子，是一栋当时就有着二十多年历史的矮房（算到今年拆迁为止，它在地球的表面一共屹立三十七年。）它的外墙是当年很流行的黄色碎玻璃样式，在顶楼镶嵌着一圈红色；外墙上被各种防盗窗、电线、空调外机和一楼住户在墙上打的洞弄得有些杂乱。它的内部结构也很有时代特色：比房间还小，不对任何窗户的客厅，和恰与之相反的房间。八十平方米的室内，能挤下四个房间、厨房、卫生间和客厅。当然，卫生间非常逼仄，只能够放下一个马桶；我们就利用马桶前面的一小点空间洗澡。虽然现在想起来会令人觉得实在不利于人方便生活，但当时我却丝毫没有这种感觉。直到我上了大学搬新家，我才体会到淋浴房很有助于提高生活质量。

我的房间南面是一扇很大的窗户，正对着隔壁楼的楼道，中间是一片一层高的储藏间。储藏间上面长着三两支杂草，偶尔有猫和麻雀会出现在其上。我无事做的时候，便会盯着顶上发呆，心想猫下一次会什么时候出现；或者看着对面的楼道，想着什么时候会有人下楼。恰与之相反，在北国，我的房间的窗户在其北面，对面是外墙刷成白色和橙色的教工宿舍区。每当晴天的时候，反射在白色区域的阳光可以给我们的宿舍提供唯一的直射光源，在苦寒的冬天里不会让我们望对门阳光直射而感到不悦。在不适合散步的日子里，我总是看着这栋楼，望着阳光，想象软红光里涌银山的场景。总而言之，窗户的存在给我的生活增加了一丝不一样的颜色。

# 七
春天对冬天的驱逐，是分阶段完成的。当河边的柳树已经抽穗、发芽的时候，百里外的山上仍然覆盖着厚厚的雪。我和同学去那里试图爬山的时候，就被猛烈的冰雹和湿滑的道路给劝退了。那边的人告诉我，要等到七月份的时候，积雪才能基本上融化，去登上山顶的人才会多起来。（但是那个时候，山上的温度仍然只有十几度）。我们于是只在山谷里转了一圈。当然，或许夏天的时候，又会有什么意外因素使得山顶对我们关闭；但是我们当时又怎么会知道呢？

山上的天气是变幻莫测的；猛烈的冰雹在我们拾级而上时候开始下落，但五分钟以后它就消失了，天继续蓝得和平原一样通透。我十分惊讶于这变幻莫测的天气，而我的同学却十分淡然：他十分热爱爬山（可能是他家在平原地区的缘故），已经征服了大大小小的山峰。当他在八月底的长白山脚下因大雪而无法继续前进的时候，他就已经领悟了这个道理。而我最后接受了这个道理，是在半年以后的星岛：当在我走在路上时，大雨突然降下，而我狼狈地去找避雨场所时，它却又突然停下了，我才理解天气的变化也是一种自然的道理。

就像天气的变化无法预测一样，我在来到星岛之前，也无法预测这一事件。两年前，我还以为我会在美国度过无法回国的至少五年；半年前的冬日，我还沉浸在我被迫延期毕业的想象中。或许在另一个世界线里，我正在伦敦的一座老旧木制房子里烧火取暖；而在另一种情况下，我在波茨坦的路上看树叶纷纷掉落，心想这里和近畿也没什么区别。而进入这些世界的钥匙，就掌握在这个夏天的我的脑海里。

# 八
在星岛的一条主要河之口，一座新近建成的桥飞跃两岸。往里走没几步，河的宽度就收窄，桥也多了起来，而且历史也变得悠久。星岛的第一座跨河桥，限于技术因素，就是在河下游的不算宽的地方修的。后来，随着技术进步和财政充实，许多更复杂的桥和隧道也不断建起。我在河口旁的石板路上散步，头上是一轮皎洁的圆月。远处，大楼的白色外墙也在黑夜里隐去，代以五光十色的霓虹灯。它们所发出灯光的亮度，与白天反射的阳光没有很大的差别。我充满好奇得看着它们，正如从前我站在小区外的湖边看着对面岛上几家酒店的光影。在我们小区拆迁之前，它们或是倒闭，或是重新装修得我都不认识了；那从前的光影，也一起被尘封进我的回忆。现在想起来，唯一不变的或许是连接小区与那个岛的桥。它算是目前一个联系从前和现在的纽带吧。

那些已经在河上屹立了一百多年的桥，在近年加上了灯光。晚上，它们在不长的河上一字排开，在夜色中显得很壮观。我上一次见到那么多桥，是在夏天的长江上。近年技术的进步给长江沿江的城市增添了不少跨江大桥，让两岸的人不止能通过声音，也能通过方便的见面互相交流。当我站在溯江而下的游船的顶端，看着桥从船的前方逐渐接近，到从我的脸上经过，然后又在船的后方消失的时候，突然感到一丝悲戚：随着船到终点，我就将永远地离去朔方大漠，到星岛开启一段新的、未知的生活。我好不容易习惯了随风满地石乱走，也和同学混熟了，却又要失去他们；这种滋味是很难让人轻易接受的。借着若隐若现的手机信号，我登录学校的内网，划过一条又一条的“教务公告”“通知消息”，其中很多条都是关于毕业生的。这些通知我是第一次仔细阅读，但同时也是最后一次。我在京兆的夏日艳阳下几于中暑，也是第一次兼最后一次。

船到了终点以后，我在武汉停留了几日。那几天，天气忽然开始变热，我为了防止流汗而多带的若干件衣服才终于派上了用场。武汉的长江大桥，是最早修建的。我在桥上来来回回地走了很多次，在早上和晚上都拍了不少照片，妄图用它们留下大学的最后一点记忆。后来，我再打开它们的时候，发现它们实际上都很模糊；不过它们是在我最后的一个夏天里拍的，这就够了。当我登上列车回到学校，再乘坐航班回到家里，我的前程便在夏天里又进了一步。

# 九
随着时光的流逝，我经历了漫长的冬季，和短暂的春天。现在，我正在星岛度过一个久长的夏。在长久的时光里，我有时回想起往事，不禁感叹季节停止变迁让我失去了很多乐趣。未来，当我回忆星岛发生的种种，可能需要另找参照了。

几百年至几十年前，许多老前辈从我的家乡来到星岛，在他们的最后的夏天里，建起桥、楼和我正就读的学校；今年，我也前赴后继地来到这里，在他们建成的校园里，继续完成未竟的事业。艳阳、绿树、明楼、黑路，定义了夏天的景色；它为数个地方和记忆所共享，连接着它们，映在我脑中。

我被永远地困在了最后的夏天里，在经历了冬天和春天以后。

# 后记
我在完成这篇文章没多久以后，和一个师姐聊天，说：“我明年夏天要回国。”她问：“夏天具体是什么时候？我们这里都是一个季节，一下子反应不过来的。”

确实，我们正在永远的夏天之中。
